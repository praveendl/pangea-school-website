# The Pangea School Website

Official website for The Pangea School - A child care center in Lawrenceville, GA.

## About

The Pangea School provides quality early childhood education and care for children from 6 weeks to school age. We are a 3-star quality rated school committed to nurturing young minds in a safe, supportive environment.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Progressive Web App (PWA)**: Can be installed on mobile devices for an app-like experience
- **Programs**: Information about all our programs (Infant Care, Toddlers, Preschool, Pre-K, After School)
- **Testimonials**: Real reviews from parents
- **Contact Information**: Easy access to phone, email, and location with Google Maps integration

## Programs Offered

- **Infant Care** (6 weeks - 12 months)
- **Toddlers** (1 - 2 years)
- **Preschool** (3 - 4 years)
- **Pre-K** (4 - 5 years)
- **After School** (School-age children)

## Technology Stack

- HTML5
- CSS3
- JavaScript
- Progressive Web App (PWA)
- Service Worker for offline functionality

## Installation as PWA

### On iPhone:
1. Open the website in Safari
2. Tap the Share button
3. Scroll down and tap "Add to Home Screen"
4. Tap "Add"

### On Android:
1. Open the website in Chrome
2. Tap the menu (three dots)
3. Tap "Add to Home Screen"
4. Tap "Add"
